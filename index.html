HTML Tags

Basics

Tag Name Description
<head> Head Acts as a container for all the stuff you want to include on the HTML page that isn't the content you are showing to your page's viewers.
<body> Body This contains all the content that you want to show to web users when they visit your page.
<h1>-<h6> Headings HTML contains 6 heading levels, h1=biggest h6=smallest
<p> Paragraph P elements are for containing paragraphs of text
<a> Anchor Creates a Link
<button> Button Creates a Button
<img> Image In order to put a simple image on a web page, we use the img element.

Source: MDN Web Docs: HTML Basics

Table tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Source: MDN Web Docs: Tables

Terminal

Tag Name Description

cd Call Directory Calling up a Directory.
pwd Print Working Directory Prints the actual path of the working Direcory.
mkdir Make Directory Creates a new Directory.
touch New File Creates a new file.
clear Clear Clears Terminal.
